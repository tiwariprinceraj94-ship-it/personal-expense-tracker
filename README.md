# 💰 Personal Expense Tracker 

---

## 📌 Project Overview

The **Personal Expense Tracker** is a powerful and user-friendly command-line application developed using Python.
It is designed to help users efficiently manage their daily financial activities by tracking income and expenses, analyzing spending patterns, and maintaining budget discipline.

This project demonstrates the use of modern Python programming concepts such as file handling, data structures, object-oriented programming, and data visualization.

---

## 🎯 Objective

The main objective of this project is to:

* Track daily income and expenses
* Store financial data permanently
* Provide meaningful summaries and insights
* Help users improve financial management
* Demonstrate real-world Python application development

---

## 🚀 Key Features

### ✅ 1. Transaction Management

* Add income and expense records
* Categorize transactions (Food, Travel, Rent, etc.)
* Store description and timestamp
* Unique transaction ID using UUID

---

### 📊 2. Financial Summary

* Monthly income calculation
* Monthly expense calculation
* Net savings calculation
* Savings rate percentage

---

### 📂 3. Category-wise Analysis

* Track expenses per category
* Identify high-spending areas
* Helps in financial decision making

---

### 💡 4. Budget Management

* Set monthly budget for categories
* Automatic warning at 80% usage
* Alert when budget is exceeded

---

### 🔍 5. Search Functionality

* Search transactions by category
* Quickly find specific records

---

### 🗑️ 6. Delete Transactions

* Remove incorrect entries
* Maintain clean and accurate data

---

### 📈 7. Data Visualization

* Pie chart representation of expenses
* Visual understanding of spending distribution
* Implemented using Matplotlib

---

### 📁 8. Data Export

* Export data to CSV file
* Useful for Excel analysis and reporting

---

### 💾 9. Persistent Storage

* Data stored in JSON format
* Automatically loads previous records
* Ensures no data loss

```bash

```

### Step 2: Install Dependencies

```bash
pip install pandas matplotlib
```

### Step 3: Run Program

```bash

```
